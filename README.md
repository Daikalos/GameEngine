# Simple Game Engine

2D game engine that is based on SFML containing a variety of functionalities for expanded usage. Goal is to then to create a game using this engine.

- Application
- Physics
- Collisions
- Animations
- UI
- Entities
- StateStack
- Networking
- Audio
- InputHandler
- ResourceManager
- SpriteBatch
- ...
